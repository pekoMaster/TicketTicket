## 設計規範

### 表格樣式
- 使用 `table` 元素，支援響應式
- 行高：48px
- 斑馬紋：奇偶行不同背景色
- 滑鼠懸停：淺色高亮

### 操作按鈕
- 編輯：藍色圖示 (Pencil)
- 刪除：紅色圖示 (Trash2)
- 封鎖：橙色圖示 (Ban)
- 解除：綠色圖示 (CheckCircle)

### 狀態標籤
- 開放中：綠色背景
- 已配對：藍色背景
- 已關閉：灰色背景
- 已封鎖：紅色背景
